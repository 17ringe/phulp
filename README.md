# phulp

The task manager for php

### Documentation

#### Install

```bash
$ composer require reisraff/phulp:dev-master
```

### Contributors Guide

#### Clone

```bash
$ git clone git@github.com:reisraff/phulp.git
$ cd phulp
$ composer install
```

#### Tests

First install the dependencies, and after you can run:

```bash
$ bin/phing
```
